Пример (CTX): Шаг 1: X — draft — очистить черновик Шаг 2: W — input — записать входной запрос пользователя Шаг 3: W — context — записать нумерованный контекст по Prompt1Context Режим: SILENT

C. Краткий справочник сущностей
